# KoraGrid Africa Website

Enterprise-grade AI solutions for Africa's most critical institutions.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Reddit Sans** - Primary typeface (headlines)
- **Satoshi** - Secondary typeface (body copy)

## Project Structure

```
KORA/
├── src/
│   ├── components/
│   │   └── Header.jsx          # Top navigation header
│   ├── App.jsx                 # Main app component
│   ├── main.jsx                # Entry point
│   └── index.css               # Global styles + Tailwind
├── KoraGrid - Website Files copy/
│   ├── Images/                 # Photography assets
│   ├── Fonts/                  # Reddit Sans & Satoshi fonts
│   ├── KoraGrid Africa - Logo Assets 2/

└── package.json
```

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build for Production

```bash
npm run build
```
