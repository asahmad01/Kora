## Typography

### Primary Typeface: Reddit Sans

- **Usage**: Headlines, titles, hero text, emphasis
- **Weights**: Bold (700)
- **Character**: Friendly, approachable, modern with rounded letterforms
- **Files**: Available in `/Fonts/Reddit_Sans Primary/`

### Secondary Typeface: Satoshi

- **Usage**: Body copy, paragraphs, descriptions, UI elements
- **Weights**: Light (300), Regular (400), Medium (500), Bold (700), Black (900)
- **Character**: Clean, readable, professional with slightly softer feel
- **Files**: Available in `/Fonts/Satoshi_Complete Secondary/`
- **Web CSS**: `/Fonts/Satoshi_Complete Secondary/Fonts/WEB/css/satoshi.css`

### Typography Hierarchy

```
Headline: Reddit Sans Bold
Subline: Reddit Sans Bold (smaller)
Body copy: Satoshi Regular/Medium
Annotation: Satoshi Regular (small)
Button: Satoshi Medium
```

## Logo Usage

### Variants

1. **Primary Logo**: Full wordmark "KoraGrid" + Icon + "Africa" subtitle
   - Minimum size: 48px height
   - Use in standard contexts (websites, documents, presentations)

2. **Icon Only**: Circular target/grid icon
   - Minimum size: 24px height
   - Use in constrained spaces (favicons, social media avatars, mobile apps)

### File Formats

- **SVG**: Preferred for web and scalable applications
- **PNG**: For raster applications
- **JPEG**: For photos/print where needed

### Clear Space

- Maintain clear space equal to the height of the icon around the logo
- No other elements should encroach on this space

### Logo Colors

- Primary: Deep Cove (#071844) on light backgrounds
- Reversed: White (#FFFFFF) on dark backgrounds
- Never use other colors or gradients

## Usage Guidelines

### Do's
✓ Use brand colors consistently
✓ Maintain proper logo clear space
✓ Use approved typefaces
✓ Keep messaging clear and direct
✓ Show African context and pride

### Don'ts
✗ Don't stretch or distort the logo
✗ Don't use unapproved colors
✗ Don't angle or rotate the logo
✗ Don't add effects (gradients, shadows, outlines)
✗ Don't use jargon or overly technical language
